riseclipse_validator_scl module

class riseclipse_validator_scl.RiseClipseValidatorSCL[source]

Bases: RiseClipseValidator

This class is a launcher for the RiseClipseValidatorSCL.jar tool.

It offers specific options for this validator.

Note

The script riseclipse_validator_scl.py can also be executed directly, it allows to donwload or update the validator jar file needed for this API.
